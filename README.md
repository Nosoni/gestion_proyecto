## Available Scripts
In the project directory, you can run:

### `npm install`
Install all dependencies from your current project. You have to run this only when you clone the project.

### `npm start`
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.

### `./database/postgrest conexiondb.conf`
Runs the web server turning the PostgreSQL database directly into a RESTful API. 
More information: [https://postgrest.org/en/v7.0.0/]

## Database Schema
Ir a la carpeta DATABASE. Encontrarás el archivo con el esquema de la BD.
Pero ojo, deberás tener cuidado al actualizar tu base de datos local.
